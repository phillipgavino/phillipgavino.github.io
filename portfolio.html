Portfolio

Check out some examples of my work below

6-DOF Stewart Platform: Design and Control

End-to-end development of a 6-DOF Stewart platform, encompassing actuator configuration, inverse kinematics, and real-time closed-loop control using vision-assisted feedback and encoder sensing. The system integrates camera-based pose estimation, constrained trajectory generation, and robust embedded control to achieve precise, stable motion under hardware and workspace limits.

3D Printer Modified Block Placement Robot

Development of a 3D-printed block placement robot: from STL geometry processing and heuristic layer-wise planning to G-code translation for execution on a modified FDM 3D printer. Implemented custom search algorithms for efficient, optimal block placement within each layer.

Engineering Design Theory: Reverse Engineered Massage Gun

Applied functional decomposition, system modeling, and iterative prototyping to evaluate product performance. This reverse engineering project emphasized thoughtful design analysis, customer needs mapping, and hands-on validation.

Robot Mechanism Design Project: Latte Art Robot

Designed and built a 1-DOF latte art robot capable of drawing heart shapes in foam. Integrated mechanism design, kinematic analysis, and precision control to achieve repeatable robotic artistry.

Precision Machine Design: Precision Tabletop Lathe

Developed a precision tabletop lathe as part of a machine design course. The project involved tolerance stack-up, FEA error modeling, and manufacturability analysis, resulting in a compact lathe designed for micron-level accuracy.

Senior Design: Sensor Bot for Evaluating Sensor Suites

Capstone project sponsored by Tailos Inc. Designed and retrofitted a modular sensor mounting platform on autonomous robots to test and validate navigation and perception sensor configurations for real-world R&D use.
